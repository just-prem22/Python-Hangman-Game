# Python-Hangman-Game
This Hangman game randomly selects a word from a list, and the player must guess the word one letter at a time. For every incorrect guess, the player loses a life and the hangman figure gradually appears. The game ends when the player either correctly guesses the entire word or runs out of lives.
---
## Overview
This is a simple Hangman game implemented in Python. The player must guess a hidden word by suggesting letters one at a time. Each incorrect guess reduces the player’s lives, and the game ends when the player either guesses the entire word correctly or runs out of lives.

The game uses ASCII art to visualize the hangman as the player loses lives.
